# TRAE-CN 文档生成器

此技能用于生成 TRAE-CN 更新后的规则、技能设置和高效方法的完整文档。

## 功能概述

本技能提供以下文档生成能力：

1. **规则更新文档** - 生成 TRAE-CN 最新规则和更新内容
2. **技能设置文档** - 生成技能配置和使用说明
3. **高效方法指南** - 提供最佳实践和优化建议
4. **快速参考指南** - 创建速查表和操作流程

## 使用场景

在以下情况下调用此技能：

- 用户请求生成 TRAE-CN 规则文档
- 用户需要技能设置说明
- 用户询问高效方法或最佳实践
- 用户需要快速参考指南
- 用户要求创建技能相关文档

## 文档结构

生成的文档应包含以下部分：

### 1. 规则更新说明

- 更新时间戳
- 主要变更内容
- 向后兼容性说明
- 迁移指南

### 2. 技能设置

- 技能配置参数
- 触发条件
- 使用示例
- 注意事项

### 3. 高效方法

- 工作流程优化
- 快捷键使用
- 常见问题解决
- 性能优化建议

### 4. 快速参考

- 命令速查表
- 快捷键对照
- 常见问题 FAQ
- 故障排除指南

## 输出格式

文档应以 Markdown 格式输出，包含：

- 清晰的标题层级
- 代码示例块
- 表格化数据
- 警告和提示框

## 质量标准

生成的文档应满足：

- ✅ 内容完整准确
- ✅ 结构清晰易读
- ✅ 包含实用示例
- ✅ 语言简洁专业
- ✅ 格式统一规范
