## 概述

Figma Remove BG是一个强大的Figma插件，用于从图片中移除背景。它支持多种背景移除方法，包括AI智能移除、颜色阈值移除、边缘检测移除和手动选择移除，帮助设计师快速处理图片。

## 功能特性

### 背景移除方法

- AI智能移除 - 使用AI算法智能识别并移除背景
- 颜色阈值移除 - 基于颜色阈值移除背景
- 边缘检测移除 - 使用边缘检测算法移除背景
- 手动选择移除 - 手动选择要移除的背景区域

### 背景颜色选项

- 透明背景 - 移除背景后使用透明背景
- 白色背景 - 移除背景后使用白色背景
- 黑色背景 - 移除背景后使用黑色背景
- 自定义颜色 - 移除背景后使用自定义颜色

### 批量处理

- 批量移除多张图片的背景
- 统一处理参数
- 批量预览结果
- 批量导出处理后的图片

### 导出功能

- 导出为PNG格式
- 导出为JPG格式
- 导出为SVG格式
- 导出为WEBP格式

### 预览功能

- 实时预览背景移除效果
- 对比原始图片和处理后图片
- 调整参数后即时预览

## 安装

1. 在Figma中打开插件管理器
2. 点击"Plugins" > "Development" > "Import plugin from manifest..."
3. 选择`manifest.json`文件
4. 插件将自动安装到Figma中

## 使用方法

### 移除单张图片背景

1. 在Figma中选择要处理的图片
2. 打开Figma Remove BG插件
3. 选择背景移除方法（AI智能、颜色阈值等）
4. 选择背景颜色选项（透明、白色、黑色或自定义）
5. 点击"移除背景"按钮
6. 查看处理结果

### 预览背景移除效果

1. 选择要处理的图片
2. 在插件中选择移除方法和背景颜色
3. 点击"预览结果"按钮
4. 查看预览效果
5. 根据需要调整参数

### 批量处理多张图片

1. 在Figma中选择多张图片
2. 在插件中选择移除方法和背景颜色
3. 点击"批量处理"按钮
4. 等待批量处理完成
5. 查看处理结果

### 导出处理后的图片

1. 选择要导出的图片
2. 在插件中选择导出格式（PNG、JPG、SVG、WEBP）
3. 点击对应的导出格式按钮
4. 图片将自动下载

### 撤销背景移除

1. 如果对处理结果不满意
2. 点击"撤销"按钮
3. 图片将恢复到原始状态

## 背景移除方法

### AI智能移除

使用先进的AI算法智能识别前景和背景，自动移除背景。适合复杂背景和精细边缘。

**优点：**

- 精度高
- 适合复杂背景
- 自动化程度高

**缺点：**

- 处理时间较长
- 需要网络连接

### 颜色阈值移除

基于颜色相似度移除背景，适合纯色或简单渐变背景。

**优点：**

- 处理速度快
- 适合纯色背景
- 离线处理

**缺点：**

- 精度中等
- 不适合复杂背景

### 边缘检测移除

使用边缘检测算法识别物体边缘，移除背景。适合轮廓清晰的物体。

**优点：**

- 适合轮廓清晰的物体
- 处理速度中等
- 离线处理

**缺点：**

- 精度中等
- 不适合模糊边缘

### 手动选择移除

手动选择要移除的背景区域，精确控制移除范围。

**优点：**

- 精度最高
- 完全可控
- 适合特殊需求

**缺点：**

- 需要手动操作
- 耗时较长

## 配置

插件支持可配置的设置：

- **默认移除方法**：选择默认背景移除方法
- **默认背景颜色**：设置默认背景颜色
- **导出质量**：设置导出图片质量
- **批量处理限制**：设置批量处理的最大数量

设置可在插件UI中访问。

## API参考

### 消息类型

插件使用以下消息类型与UI通信：

- `remove-background`: 移除背景
- `preview-result`: 预览结果
- `undo-removal`: 撤销移除
- `export-image`: 导出图片
- `batch-remove`: 批量移除
- `load-image`: 加载图片
- `load-config`: 加载插件配置
- `save-config`: 保存插件配置
- `close`: 关闭插件
